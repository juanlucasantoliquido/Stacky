# Restricciones del proyecto STACKY-AGENTS

- Foco en tareas acotadas: un modulo o un archivo por vez.
- Si la tarea requiere leer mas de 5 archivos, dividirla en subtareas.
- Evitar analisis globales del repo completo en un solo turno.
- Si algo aplica a todos los agentes del ecosistema, proponer moverlo a la capa core.
- Ventana de contexto limitada a 16k tokens: priorizar precision sobre exhaustividad.
