# repo-onboarding

Objetivo: orientarse rapidamente en el repo STACKY-AGENTS.

Pasos:
1. Leer el README o CLAUDE.md si existe.
2. Listar los directorios de primer nivel para entender la estructura.
3. Identificar el entry point principal (index.ts, main.ts, o similar).
4. Verificar conexion al modelo local antes de continuar.
