Fix requerido — CA-08

Archivo: trunk/OnLine/AIS.PR.UI.Web.Controls/AISMenu.cs

Línea ~584:

// ❌ Actual (incorrecto — visible para todos):
if (!EsJudicial)
{
    writer.Write(...Promesas a Vencer en 7 días...);
}

// ✅ Requerido (visible solo si instancia Pacífico):

    writer.Write(...Promesas a Vencer en 7 días...);
}

Nuevo método privado a implementar en AISMenu.cs (según spec del analista técnico):

// ADO-116 | Verifica si el usuario conectado pertenece a la instancia Pacífico (PEEMPRESA='0010')
private bool EsInstanciaPacifico(string codUsuario)
{
    try
    {
        RSFac.Agenda agenda = new RSFac.Agenda();
        return agenda.EsInstanciaPacifico(codUsuario);
    }
    catch { return false; }
}

El pass-through en RSFac/RSBus/RSDalc ya fue previsto en el análisis técnico (query: RUSUPERF JOIN RPERFEMP WHERE PEEMPRESA='0010').
