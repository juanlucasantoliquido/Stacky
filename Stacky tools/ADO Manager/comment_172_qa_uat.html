Si bien el DEFAULT protege los casos normales, sería más robusto agregar ALTER TABLE dbo.RDIRE ALTER COLUMN DTOFICIAL VARCHAR(1) NOT NULL una vez confirmado que todos los registros históricos tienen valor (post-migración CA-09). Impacto: bajo. No bloquea el release.
