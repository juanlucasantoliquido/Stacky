1. TRADUCCIÓN FUNCIONAL → TÉCNICA

Requerimiento funcional (resumen)

RF-005 pide coherencia entre criterios de filtro y columnas visibles en Agenda Personal. Para los nuevos campos de Pacífico ya agregados al filtro (Débito Automático, Corredor, Nombre de Cliente, RUC), la lista debe reflejar al menos los datos equivalentes visibles en resultado.

Solución técnica propuesta

La UI de lista en FrmAgenda está definida por columnas fijas de AISGridView y no se autoalinea con filtros. Se debe extender SetColumnasGrillaAgendas() con las columnas faltantes y ampliar los SELECT de RSDalc.Agenda.Consultar() (AGENDAAUT y AGENDAUSU) para devolver los campos requeridos. No requiere cambio en firmas RSFac/RSBus ni DDL: los filtros ya están implementados y las columnas BD existen.

Flujo actual del sistema (cómo funciona HOY)

  1. Usuario ejecuta búsqueda en FrmAgenda.aspx con 11 filtros (incluyendo Débito Auto/Corredor/Nombre/RUC).
  2. FrmAgenda.FiltrarGrilla() invoca RSFac.Agenda.Consultar() y carga GridAgendaUsu/GridAgendaAut.
  3. SetColumnasGrillaAgendas() solo muestra Cliente, Lote, Fecha, Hora, Recomendación, Nivel Mora, Prima, Moneda.
  4. RSDalc.Agenda.Consultar() filtra por RUC/Corredor/Débito, pero no retorna esas columnas en SELECT.

Flujo propuesto (cómo debe funcionar DESPUÉS)

  1. (sin cambio) Se mantienen los filtros actuales.
  2. [CAMBIO] SetColumnasGrillaAgendas() agrega CLNUMDOC, OGCORREDOR y OGDEBAUT.
  3. [CAMBIO] RSDalc.Agenda.Consultar() incorpora en ambos SELECT: CLNUMDOC, OGCORREDOR y cálculo OGDEBAUT (S/N, NULL→N).
  4. La grilla queda consistente con los filtros visibles para RUC, Corredor y Débito Auto.

Queries para obtener datos de prueba

-- Candidatos de agenda pendiente
SELECT TOP 20 A.AGLOTE, A.AGFECREC, A.AGHORAREC,
       O.OGTIENEDEBITOAUTO, O.OGCORREDOR,
       C.CLNUMDOC, C.CLAPELLIDO1, C.CLNOMBRE
FROM RAGEN A
INNER JOIN RLOTE L ON L.LOCOD = A.AGLOTE
INNER JOIN ROBLG O ON O.OGLOTE = L.LOCOD AND O.OGLIDER='1'
INNER JOIN RCLIE C ON C.CLCOD = L.LOCOD
WHERE A.AGHECHO='P'
ORDER BY A.AGFECREC DESC, A.AGHORAREC DESC;

-- Distribución para Débito/Corredor
SELECT COUNT(1) AS TotalPendientes,
       SUM(CASE WHEN O.OGTIENEDEBITOAUTO='S' THEN 1 ELSE 0 END) AS DebitoSi,
       SUM(CASE WHEN O.OGTIENEDEBITOAUTO='N' THEN 1 ELSE 0 END) AS DebitoNo,
       SUM(CASE WHEN O.OGTIENEDEBITOAUTO IS NULL THEN 1 ELSE 0 END) AS DebitoNull,
       SUM(CASE WHEN O.OGCORREDOR IS NOT NULL AND LTRIM(RTRIM(O.OGCORREDOR))<>'' THEN 1 ELSE 0 END) AS ConCorredor
FROM RAGEN A
INNER JOIN RLOTE L ON L.LOCOD=A.AGLOTE
INNER JOIN ROBLG O ON O.OGLOTE=L.LOCOD AND O.OGLIDER='1'
WHERE A.AGHECHO='P';
